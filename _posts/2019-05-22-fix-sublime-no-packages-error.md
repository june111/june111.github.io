# Sublime 无法使用 Package Control 的解决方法

问题：使用 `Package Control: Install Package` 后，弹出提示 “There are no packages available for installation”

解决方法：

1. `Package Control: Add Channel`
2. 输入 https://github.com/wilon/sublime/raw/master/download/channel_v3.json
3. 显示 Channel added 表示成功了
